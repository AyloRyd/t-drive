# t-drive

A file storage app made with:

- TypeScript
- React
- Next.js
- SingleStore
- Clerk
- Uploadthing
- Tailwind CSS
- Shadcn UI

Deployed on Vercel: [check it out!](https://t-drive-cloud.vercel.app)
